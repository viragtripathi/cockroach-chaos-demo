💡 How It Works

Network Partition Mode: Simulates network failures via Toxiproxy. HAProxy detects unhealthy nodes and instantly reroutes traffic to healthy regions. CockroachDB nodes stay running - check the admin console!

Node Failure Mode: Actually stops Docker containers (docker stop). This demonstrates true resilience - your app retries connections and HAProxy redirects to available nodes. Recovery takes longer as containers restart.
